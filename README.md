# Интернет-магазин карандашей

Это учебный проект — интернет-магазин карандашей.  
При разработке использовались следующие технологии: 
- TypeScript + JavaScript
- React + Redux Toolkit + Next
- Vite
- HTML и SCSS-модули + БЭМ
- GIT + GitHub
- PHP + MySQL
- npm
- DevTools
- Docker

Сайт адаптирован под любую ширину экрана и любой браузер, а также учитывает принципы цифровой доступности (accessibility), что обеспечивает удобство использования для всех пользователей, включая людей с ограниченными возможностями. 
В проекте реализованы:  
- товары с возможностью добавления в корзину и избранное  
- фильтрация и сортировка товаров  
- выбор категорий и поиск товаров  
- пагинация  
- регистрация, вход и выход из аккаунта, редактирование и удаление аккаунта
- смена темы  
- слайдер с рекламой  
- горизонтальная прокрутка рекомендуемых товаров
- оформление заказов и пополнение баланса
- админ-панель

Проект находится в активной стадии разработки, следующим шагом в развитии проекта является глубокая работа над архитектурой, включающая рефакторинг монолитных компонентов и отделение логики от UI, разработка нового функционала и внедрение unit-тестирования с Jest.

## Запуск проекта через Docker

Проект полностью контейнеризирован и может быть запущен с помощью Docker и Docker Compose.  
Этот способ позволяет запустить все компоненты (PHP, MySQL, Next.js) одной командой без необходимости настраивать окружение вручную.

### Предварительные требования

- Установленный **Docker Desktop** (для Windows) или **Docker Engine** (для Linux/Mac)
- Включённый **WSL2** (для Windows)

### Запуск проекта

1. **Клонируйте репозиторий** (если не сделали ранее).
2. **Откройте терминал** в корневой папке проекта.
3. **Выполните команду:** `docker-compose up --build`.
Эта команда:
- Соберёт образы для PHP и Next.js
- Загрузит образ MySQL
- Запустит три контейнера: shop-mysql, shop-php, shop-nextjs

4. После успешного запуска откройте в браузере: http://localhost:3001

### Команды для управления Docker

| Команда | Описание |
|---------|----------|
| `docker-compose up --build` | Собрать и запустить проект (с пересборкой образов) |
| `docker-compose up -d` | Запустить проект в фоновом режиме |
| `docker-compose down` | Остановить все контейнеры (данные БД сохраняются) |
| `docker-compose down -v` | Остановить контейнеры и удалить volume с БД (при следующем запуске база восстановится из дампа) |
| `docker-compose logs -f` | Показать логи всех контейнеров в реальном времени |
| `docker ps` | Показать список запущенных контейнеров |

### Особенности работы с Docker
- Первый запуск может занять 5–10 минут — Docker скачивает образы MySQL, Node.js и PHP.
- Последующие запуски происходят быстрее.
- База данных автоматически восстанавливается из дампа pencilShopDump.sql при первом запуске.
- Изображения товаров хранятся в папке uploads/tovar/ и монтируются в контейнер.
- Все данные БД сохраняются между перезапусками (если не использовать флаг -v).

### Остановка проекта
- Команда: `docker-compose down`
- Если нужно полностью очистить все данные (включая БД): `docker-compose down -v`

## Локальный запуск проекта (без Docker):

1. **Клонируйте репозиторий** на свой ПК.
2. **Установите зависимости** командой: `npm install`.
3. **Запуск PHP-сервера:**
   - Для локальной работы PHP-скриптов используйте расширение **PHP Server** в VS Code:
     - Выберите команду: `PHP Server: Serve project`.
     - Расширение запустит PHP-сервер на порту `3000`.
   - Либо без расширений выполните в терминале: `php -S 0.0.0.0:3000`.  
     *Перед этим убедитесь, что PHP установлен и добавлен в PATH (проверить можно командой `php -v`).*
4. **Настройка базы данных:**
   - Импортируйте дамп `pencilShopDump.sql`.  
     Путь к файлу: `backend/MySQL/pencilShopDump.sql`.
   - Используйте MySQL Workbench, phpMyAdmin или командную строку.
5. **Настройка подключения к БД:**
   - В папке `backend/PHP/config` найдите файл `db.php`.  
     Там указаны параметры подключения (хост, пользователь, пароль, название базы).
6. **Сборка и запуск фронтенда:**
   - Соберите проект: `npm run build`
   - Запустите в режиме предпросмотра: `npm run start`
   - Или сразу запустите в режиме разработки: `npm run dev`
7. **Откройте в браузере:** `http://localhost:3001`
8. **Доступ с других устройств (в одной сети Wi-Fi):**
   - Узнайте IPv4-адрес вашего ПК (он отображается в терминале после сборки).
   - На другом устройстве откройте: `http://<IPv4-адрес>:3001`

> **Важно:** Перед запуском убедитесь, что PHP-сервер запущен и подключён к базе данных.

9. **Доступ к админ-панели:**
   - Войдите в аккаунт администратора.  
     В таблице `users` в колонке `is_admin` должно быть `1`.
